# Tienda_videojuegos
Este proyecto analiza datos históricos de ventas de videojuegos para la tienda online Ice, con el objetivo de identificar patrones que permitan determinar qué factores influyen en el éxito comercial de un videojuego. Los resultados buscan apoyar la planificación de campañas publicitarias y la identificación de proyectos prometedores en 2017.
