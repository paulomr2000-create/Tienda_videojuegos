# Tienda_videojuegos
Este proyecto analiza datos históricos de ventas de videojuegos para la tienda online Ice, con el objetivo de identificar patrones que permitan determinar qué factores influyen en el éxito comercial de un videojuego. Los resultados buscan apoyar la planificación de campañas publicitarias y la identificación de proyectos prometedores en 2017.
Objetivos

Analizar tendencias históricas en ventas de videojuegos.

Identificar plataformas y géneros más rentables.

Evaluar el impacto de las reseñas de usuarios y críticos en las ventas.

Analizar diferencias de mercado entre regiones.

Validar hipótesis estadísticas relacionadas con calificaciones de usuarios.

Herramientas utilizadas

Python

Pandas

NumPy

Matplotlib / Seaborn

Resultados Esperados

El proyecto permite identificar tendencias del mercado de videojuegos, detectar oportunidades comerciales y optimizar estrategias de marketing basadas en datos históricos y análisis estadístico.
